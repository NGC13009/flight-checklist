# 飞行检查清单
这里提供了关于 c172 和 a320/a321 的飞行检查清单中文版，翻译自[mentoneflyingclub](https://mentoneflyingclub.org/)和[flyuk](https://flyuk.aero/).

## 支持机型

### 塞斯纳 172（C172）

塞斯纳 172 是全球最受欢迎的单引擎飞机之一。本仓库包含详细检查清单，涵盖：

- 飞行前检查
- 发动机启动程序
- 起飞配置
- 飞行中操作
- 着陆程序
- 飞行后检查

### 空客 A320 / A321

空客 A320 系列是应用最广泛的商用窄体客机。包含的检查清单包括：

- 飞行前准备
- 驾驶舱飞行前检查
- 发动机启动与滑行程序
- 起飞配置
- 爬升程序
- 进近与着陆
- 飞行后检查

## 使用说明

从`pdf`文件夹下下载两个pdf，或克隆源代码到本地，然后在`pdf`文件夹下找到编译好的内容。

所有内容已经被压缩到刚刚好一张纸正反两页。（一些地方写不下，就把英文省略了）

## 编译

使用texlive2025环境，XeLaTeX分别编译tex即可。

## 安全声明

注意！仅供游戏使用，如果应用于任何真实飞行则可能构成违法或犯罪，甚至造成事故！
本检查清单仅用于信息参考和训练目的。实际飞行操作中，务必查阅官方飞机操作手册和制造商文档。
