## 2. 推出与发动机启动 (Pushback & Engine Start)

### 2.1 推出前准备

- FOB 确认一致，起飞数据不变
- 断开外部电源，撤离外部电源
- 联系机坪 (_____ MHz)：（航班号）_****，（机场）****_，是否可以推出开车
- 证实可以推出开车，沿红线

### 2.2 启动前检查单 (Before Start Checklist)

- 停留刹车 … **设置** (SET)
- 起飞/推力调定：V1 = _____ kt  
  VR（抬轮速度） = _____ kt  
  V2（安全速度） = _____ kt  
  灵活温度（FLX） _____°C
- 侧窗 … **关闭**
- 信标灯 (Beacon) … **接通**
- **启动前检查单完成**
- 地面呼叫：“可以松刹车？” → “压力零，刹车已松开”
- 推出红线 → “收到，红线推出”
- “可以起动发动机吗？” → “可以起动”

### 2.3 发动机启动

- 启动顺序：2 发 → 1 发
- 地面呼叫：“设置刹车” → 停留刹车设置，压力正常 → “刹车已设置”
- “可以脱开” → 脱开左侧 [] / 右侧 [] 手势

### 2.4 启动后检查单 (After Start Checklist)

- 防冰 (Anti-Ice) … **OFF**
- ECAM 状态 … **已检查**
- 俯仰配平 (Pitch Trim) … **_____° UP** (28%)
- 方向舵配平 (Rudder Trim) … **中立位**
- 滑油压力 … **正常**
- 液压压力 … **正常**

- **启动后检查单完成**

## 9. 着陆 (Landing) (记忆动作)

### 9.1 最后进近与着陆

- AP OFF, "自动驾驶断开, 我来操作"

- 100 ft 以上 → 继续 / 复飞
- 300 ft → MINIMUM → 200 ft → 100 ft → 跑道可见
- 接地: 扰流板 (SPOILERS) 升起，反推 (REVERSERS) 绿色，减速
- 70 kt → 反推慢车
- 40 kt → 人工刹车
- 快速脱离道脱离
- 脱离路线: __________________________________________

### 9.2 着陆后滑行
